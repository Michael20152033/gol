Cryptoestate - online monopoly
Browser games and games for social network
https://cryptoestate.goldapp.ru/
Challenge

It was decided to develop the game for PC and smartphone browsers in order to reach the maximum number of users, without forcing them to install the game on a device as an application.

The TRON blockchain was chosen for the development of a game token to minimize the transfer fees between wallets.

Solution

It was necessary to develop an online "Monopoly" game with the ability to play with cryptocurrency, win and withdraw the earned funds. It was also necessary to develop a custom game token

Impact

As a result, the "Cryptoestate" browser game was developed with its own CRES token (TRON blockchain). Integration with etherapi.net and tronapi.net. The ability to play with a bot or other players. The ability to follow the game as a spectator. Referral system. Educational game. Cryptolotery.

Conducting tournaments and obtaining titles. Automated withdrawal of user earned cryptocurrency.

Admin dashboard for tournament creation, withdrawal confirmations, and moderation.
